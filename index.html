iMacros Version 11.5 (Windows 64-bit and 32-bit version)

Trial версия iMacros 11.5 включает в себя возможности standart и Enterprise сборки.

Системные требования

Window 10, Windows 8 / 8,1, Windows 7, Windows XP SP3, Windows Server 2003/2008 / 2008R2 / 2012 / 2012R2 Windows 32-бит (x86) и 64-разрядный (x64). Microsoft Internet Explorer 9, 10 или 11 (для Windows XP поддержка только IE8) Mozilla Firefox версии 21 и выше (опция, только для Firefox iMacros Add-On) Google Chrome Версия 22 или выше (опция, только для iMacros Chrome Add-On) 256 МБ оперативной памяти (рекомендуется 512 МБ) 30 Мб свободного пространства на жестком диске download imacros 11.5
